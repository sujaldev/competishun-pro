# Competishun Pro

Chrome extension to help watch lectures in competishun web app.

### Features

- Keeps track of duration of lecture watched.
- Removes annoying floating watermark in lectures

### Upcoming Features (maybe)

- Youtube like keybindings
- Auto Login
