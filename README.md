# Competishun Pro

Chrome extension to help watch lectures in competishun web app.

### Features

- [x] Keeps track of duration of lecture watched.
- [x] Removes annoying floating watermark in lectures
- [x] Multiple Logins
- [ ] Youtube like keybindings
